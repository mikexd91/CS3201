===== RUN INSTRUCTIONS =====
1. Ensure that Python2.7 is installed (Windows)
2. Ensure that project has been built (SPA in debug, AutoTester in release)
3. Run the python scripts in test root folder - /Test02
	a. run_all_tests.py - runs all available tests
	b. run_core_tests.py - runs all tests for functionality
	c. run_stress_tests.py - runs tests for query optimiser
	d. /test_bonus/run_extensions_tests.py - runs all extensions tests
4. Test results are avaiable in Results folder

NOTE: For testing of cyclic calls, tests considered passed if cyclic<x>.xml is not found in results
